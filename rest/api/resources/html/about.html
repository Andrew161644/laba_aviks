Программное обеспечение для распознавания жестов в режиме реального времени

Цель данного продукта – распознавание жестов, предназначенных для управления роботом. Например, команда «стоп», отображаемая жестом

Задача заключается в том, чтобы по фото/видеопоследовательности на входе выделить и классифицировать показанный жест. Данная задача относится к области компьютерного зрения.

Практическая ценность программного обеспечения заключается в том, что, во-первых, можно распознавать жесты в режиме реального времени, а во-вторых, ПО может быть использовано в робототехнических системах для управления автоматическим устройством.
